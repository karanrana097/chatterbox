# Chatterbox

Chatterbox is a simple chat application built with Flutter and Firebase. It includes essential features like user authentication, real-time messaging, and a clean user interface.

## Features

- Welcome Screen
- Login Screen
- Registration Screen
- Chat Screen

## Getting Started

Follow these instructions to set up and run the project on your local machine for development and testing purposes.

### Prerequisites

- [Flutter](https://flutter.dev/docs/get-started/install) - Follow the official documentation to install Flutter on your machine.
- [Firebase Account](https://firebase.google.com/) - You need a Firebase account to use Firebase services.

### Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/karanrana097/chatterbox.git
   cd chatterbox
   ```

2. **Install dependencies:**

   ```sh
   flutter pub get
   ```

3. **Set up Firebase:**
   - Create a new project in the [Firebase Console](https://console.firebase.google.com/).
   - Add an Android/iOS app to your Firebase project.
   - Follow the instructions to download the `google-services.json` (for Android) or `GoogleService-Info.plist` (for iOS) file and place it in the appropriate directory of your Flutter project.

4. **Run the app:**

   ```sh
   flutter run
   ```
